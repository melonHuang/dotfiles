dotfiles
========

Configuration files for vim, bash, etc.

### How to use dotfiles

1. Get dot files
    * from git repository: `git clone git@github.com:webzhao/dotfiles.git` or
    * [download the zip archive](https://github.com/webzhao/dotfiles/archive/master.zip)

2. Install

   ```
   cd dotfiles
   sh install.sh
   ```
